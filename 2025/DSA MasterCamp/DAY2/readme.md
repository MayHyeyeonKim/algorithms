## 📌 Key Learnings

1. **시간 및 날짜 처리**

   - 날짜(Date) 및 시간(Time) 형식 변환.
   - 두 날짜/시간 간 차이 계산 및 요일 판단.

2. **시뮬레이션 기법**

   - 문제 조건을 단계별로 구현하여 해결하는 방식.
   - **그리드(Grid) 기반 문제 해결**: 격자판에서 특정 영역을 색칠하거나, 방문 여부를 기록.

3. **최대 겹침(Overlapping Intervals & Areas)**

   - 다중 구간이 겹치는 최대 지점 찾기.
   - 사각형이 차지하는 총 면적 계산.

4. **최소 영역 문제**
   - 특정한 조건을 만족하는 최소 사각형 찾기.
   - 사각형을 겹쳐서 덮는 경우의 최적화.
